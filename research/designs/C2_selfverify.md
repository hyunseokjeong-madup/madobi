# Operating manual: Decompose + Independent Self-Verification

Phase A — Solve:
1. Restate what is asked and list all constraints explicitly.
2. Decompose into sub-steps; solve each, writing intermediate results.
3. Produce a candidate answer.

Phase B — Verify (MANDATORY, use a DIFFERENT method than Phase A):
4. Re-derive the answer by an independent route (e.g. count the complementary case,
   plug the answer back into the constraints, use a different formula, or check a small base case).
5. If the two methods disagree, find the error and redo until they agree.
6. Check the answer against sanity bounds (sign, magnitude, units, format).

Only commit once an independent check agrees. Output in exactly the required format.

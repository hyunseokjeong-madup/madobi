# 광고 소재(Creative) 플레이북 — 기획 · 생성 · 점검

소재의 전 주기를 다루는 재사용 지식자산. (분석 지표는 `METRICS.md` D항 참조.)

## 1) 기획 (Planning)
브리프를 받으면 다음을 *명시적으로* 채운다 — 비면 질문한다:
- **목표/KPI**: 인지(CPM/VTR) · 유입(CTR/CPC) · 전환(CPA/ROAS) 중 무엇이 1순위인가.
- **타깃**: 페르소나, 인사이트(통점/욕구), 퍼널 단계(상단/중단/하단).
- **앵글(메시지 축)**: 혜택/문제해결/사회적증거/희소성/가격/신규성/호기심 중 1차·2차.
- **포맷/매체**: 동영상(6/15/30s)·이미지·캐러셀·UGC·플레이어블; 매체 규격(메타/유튜브/틱톡/구글).
- **후크(첫 3초/첫 줄)**: 시선 정지 요소 — 질문·숫자·반전·얼굴·텍스트 오버레이.
- **CTA**: 한 개, 명확하게.
- **가설**: "이 앵글×후크가 이 타깃의 CTR/CVR을 올린다" — 측정 가능하게.

## 2) 생성 (Generation)
- **변주 매트릭스**: 앵글 × 후크 × 포맷으로 **체계적 변주**(무작위 X). 최소 3~5안, 1변수만 바꿔 학습 가능하게.
- **카피 구조**: Hook → Value(혜택·증거) → CTA. 길이/금칙어/숫자표현(₩, %, 배수) 일관.
- **후크 공식 예**: 숫자("3일 만에 ~"), 질문("아직도 ~?"), 반전, before/after, 사회적증거("N만 사용").
- **네이밍 규칙(필수, 분석 정합성의 핵심)**: `{캠페인}_{앵글}_{포맷}_{후크}_{버전}` — 소재명만 봐도
  차원 분해가 되도록. (분석 시 차원합=총계 검산이 가능해진다.)
- **금지**: 근거 없는 과장, 미검증 수치, 규제 위반 표현.

## 3) 점검 (Review / QA) — 체크리스트
- **규격(Spec)**: 해상도/비율/용량/길이/자막 안전영역, 매체별 텍스트 비율.
- **클레임/규제**: 과장·최상급·의료/금융 표현, 비교광고 근거, 개인정보·저작권/초상권.
- **브랜드**: 로고·컬러·톤&매너·금칙어, 법적 고지(필수 표기).
- **메시지 정합성**: 후크↔본문↔CTA↔랜딩페이지 일관(메시지 매치). 불일치 시 CVR 하락.
- **수치 표현**: 모든 숫자는 출처/기준 확인 — 소재에 들어가는 수치도 `reconcile` 대상.
- **접근성**: 자막/대비/가독성.
- **A/B 설계**: 1변수 격리, 표본·기간 동일, 소표본 과해석 금지.

## 4) 기획→생성→점검→분석 루프
```
기획(가설) ─▶ 생성(변주 매트릭스) ─▶ 점검(QA 체크리스트) ─▶ 집행
     ▲                                                          │
     └──────── 분석(METRICS D: 승자/패자·피로) ◀── 정합성 검산 ─┘
```
- 매 사이클의 **승리 앵글/후크/포맷**과 실패 패턴을 **놀리지에셋에 누적**(계정별). 쓸수록 기획이 빨라진다.
